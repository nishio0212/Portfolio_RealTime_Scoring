# RealTime Scoring — 日本株リアルタイム・スコアリングシステム

> **「毎日14時30分、独自アルゴリズムが有望銘柄を自動でランキングして届けます。」**

---

## 概要

このツールは、東京証券取引所に上場する日本株を対象に、独自のテクニカル指標をもとにリアルタイムでスコアリングし、有望銘柄を自動でランキングするシステムです。

毎営業日の取引時間中に実行すると、対象銘柄の株価データを自動で収集・分析し、結果をCSVファイルとして出力します。出力されたランキングはGmailで自動送信することもできるため、**パソコン操作なしに毎日スコア結果を受け取る**運用が可能です。

設計から実装まで一貫して自動化されており、一度セットアップすれば毎日ほぼ手作業ゼロで動き続けます。

---

## 主な機能

| 機能 | 内容 |
|---|---|
| 銘柄リスト自動取得 | JPX（日本取引所グループ）から最新の上場銘柄情報を毎回自動ダウンロード |
| 株価データ収集 | Yahoo Financeから過去1年分の日足データと当日の分単位データを自動取得 |
| テクニカル指標計算 | EMA・MACD・RSI・VWAPなど主要指標を自動計算 |
| 独自スコアリング | 10種類の買いシグナル条件 ＋ 4種類のペナルティ条件 ＋ ボーナスルールで総合スコアを算出 |
| 出来高・売買代金フィルタ | 流動性の低い銘柄を自動除外（出来高300万株未満・売買代金100億円未満） |
| 時刻モード対応 | 14:30・15:00・15:15・15:30の各時点でのスコアリングに対応 |
| CSV自動出力 | スコア上位銘柄を日付・モード別にランキング形式で保存 |
| メール自動送信 | 結果CSVをGmailで指定アドレスに自動送信 |

## 使用技術

| 技術・サービス | 用途 |
|---|---|
| **Python 3.10** | システム全体の実装言語 |
| **yfinance** | Yahoo Financeから株価データを取得するライブラリ |
| **pandas / numpy** | 大量の株価データの集計・計算処理 |
| **JPX（日本取引所グループ）** | 上場銘柄の公式リスト（`data_j.xls`）をダウンロード |
| **Yahoo Finance** | 日足・分足の株価データ取得元 |
| **Gmail SMTP** | 分析結果CSVの自動メール送信 |
| **Windows タスクスケジューラ** | 毎営業日の指定時刻に自動実行するための仕組み |

## システムの仕組み

```
STEP 1  JPX銘柄リスト取得
        JPXの公式サイトから最新の上場銘柄リスト（data_j.xls）を自動ダウンロード
                ↓
STEP 2  対象銘柄データの構築
        監視リスト（input/codes.csv）の銘柄コードと照合し、
        銘柄名・市場区分・業種情報を付与したデータを生成
                ↓
STEP 3  株価データ収集 ＋ テクニカル指標計算
        各銘柄について過去1年分の日足データをYahoo Financeから取得し、
        EMA・MACD・RSI・VWAPなどの指標を自動計算
                ↓
STEP 4  スコアリング
        独自ルール（10条件＋ペナルティ＋ボーナス）でスコアを算出。
        出来高・売買代金が基準を下回る銘柄は自動除外
                ↓
STEP 5  CSV出力 ＋ メール送信
        スコア降順でランキングし output/YYYY-MM-DD/rank_{モード}.csv に保存。
        send_csv.py が Gmail で指定アドレスに自動送信
```

## 実行方法

```bash
# 基本実行（14:30モード）
python main.py 1430

# 15:00モードで実行
python main.py 1500

# バッチファイルで実行（ログ自動保存）
RealTime_scoring.bat 1430

# 結果CSVをGmailで送信
python send_csv.py 1430
```

**モードの意味：**

| モード | 説明 |
|---|---|
| `1430` | 当日14:30時点の株価でスコアリング |
| `1500` | 当日15:00時点の株価でスコアリング |
| `1515` | 当日15:15時点の株価でスコアリング |
| `1530` | 当日の終値（引け後）でスコアリング |

## ディレクトリ構成

```

│
├── main.py                   # メイン実行ファイル

├── module/                   # 処理モジュール群
│   ├── codes_data_select.py  # 銘柄データ構築

│       ├── dl_codes.py           # JPX銘柄リストDL
│       ├── yahoo_get_history.py  # 株価データ取得
│       ├── calculate/            # テクニカル指標計算（EMA/MACD/RSI/VWAP）
│       └── scoring/              # スコアリングロジック（非公開）
│
├── input/                    # 監視銘柄リスト（非公開）
├── data/                     # JPX銘柄データ（自動生成）
├── output/                   # スコアリング結果CSV（日付別）
├── logs/                     # 実行ログ
└── snapshot/                 # リアルタイム価格スナップショット
```

## 注意事項

- **スコアリングロジック（`module/function/scoring/`・`module/target_scoring.py`）は非公開です。**
  独自に設計・チューニングした売買シグナルの判定条件および点数配分が含まれるため、このリポジトリには含まれていません。

- 本ツールは情報収集・分析を目的としたものです。投資判断はご自身の責任のもとで行ってください。

- Yahoo Finance・JPXのデータ取得には外部サービスへのアクセスが発生します。各サービスの利用規約をご確認ください。

- Python 3.10 および必要ライブラリのインストールが前提となります。
